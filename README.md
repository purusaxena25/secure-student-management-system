# Secure Student Management System

A Java-based backend application developed using object-oriented programming principles.
The system is designed to manage student records securely with database integration.

## Features
- Student record management (CRUD operations)
- MySQL database integration using JDBC
- Role-based access control (RBAC)
- Input validation and secure coding practices
- Error handling and debugging for stable system behavior

## Tech Stack
- Java
- MySQL
- JDBC

## Project Type
Academic / Backend learning project

This project demonstrates backend development fundamentals, database connectivity,
and secure application design.
